## Compile
Install packages
```bash
$ pip install -r requirements.txt
```

Compile
```bash
$ populus compile
```

## Deploy
```bash
$ ./scripts/deploy.sh
```

## Running the tests

You can run the tests with:
```bash
$ pytest tests/
```
